# ЧМ-2026 · Аналитический центр ⚽

Интерактивное веб-приложение для анализа шансов сборных на чемпионате мира 2026 (США · Канада · Мексика). Один HTML-файл, без сборки и зависимостей — открывается двойным кликом.

## Возможности

- **Шансы на титул** — вероятности победы для 14 главных претендентов с рейтингом Эло.
- **Подиум фаворитов** — топ-3 с процентами.
- **Профиль команды** (по клику): обзор, тактический стиль, профиль силы по 6 параметрам, ключевые игроки с оценками, сильные/слабые стороны (SWOT) и путь к финалу.
- **Ближайшие матчи** — реальное расписание заключительных туров группового этапа (24–27 июня 2026) с прогнозом исхода **П1 / ничья / П2**, вероятным счётом и кратким пояснением.
- **Фильтры и поиск** — по фаворитам, тёмным лошадкам, континентам.
- Настоящие флаги стран (flagcdn), адаптивный дизайн, тёмная тема.

## Как запустить

Просто откройте `index.html` в браузере. Либо локальный сервер:

```bash
npx serve .
```

## Обновление матчей

- **Статусы и время** обновляются автоматически: каждый матч показывает «предстоит» (с обратным отсчётом), «🔴 LIVE» или «✓ завершён» в зависимости от реального времени. Список сам пересортировывается (идущие/ближайшие — наверху). Авто-обновление каждые 30 сек + кнопка «🔄 Обновить».
- **Реальный счёт.** Файл [`results.json`](results.json) — необязательный источник настоящих результатов. Пока он пуст, сайт показывает прогноз модели. Чтобы вывести реальный счёт, добавьте записи:
  ```json
  { "updated": "2026-06-24T23:00:00Z",
    "results": [ { "a": "sui", "b": "can", "ga": 1, "gb": 1, "status": "finished" } ] }
  ```
  `status`: `"live"` или `"finished"`; `minute` — минута для LIVE. Коды команд — как в `index.html`. После пуша сайт подхватит данные сам (нужен запуск через веб-сервер, не `file://`).
- Для **полностью автоматического** обновления результатов нужен внешний спортивный API или плановый агент (GitHub Action / cron), который будет писать `results.json`. Готов настроить отдельно.

## Методология

Прогноз матча — **модель Пуассона**: из рейтинга Эло (с поправкой на фактор своего поля у хозяев) рассчитываются ожидаемые голы (xG) каждой команды, затем по распределению Пуассона — вероятности всех счетов. Отсюда исход (П1/Х/П2), самый вероятный счёт и топ альтернативных счетов. Шансы на титул — отдельная экспертная оценка на базе рейтинга и силы состава.

## Дисклеймер

Расписание матчей — реальное. Вероятности — это смоделированная оценка (Эло + форма + сила состава), **не гарантия результата и не призыв к ставкам**. Реальный футбол непредсказуем.
